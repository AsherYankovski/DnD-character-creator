# Создавалка персонажа для DnD
## Зачем
Ради атмосферного, старенького опыта гиковских увлечений. Ну и ради практики изучения языка C. Ну и конечно же ради большой и чистой любви к ролёвкам.\
В целом, это может иметь и маленький практический смысл, если Вы хотите создавать себе персонажей в долгих поездках или отдалённых местах без интернета :)
## Основной функционал
Пока только планируемый, конечно же, но всё же.\
**Как минимум**, стоит написать сам функционал создания персонажа с выбором рассы, класса и т.п., разумеется прикрутив к нему автоматичесое заполнение статов и выбор снаряжения.\
**Как планы на очень далёкое будущее**, прикрутить к этому всему возможность выкачивать откуда то инфу о классах и прочем, да ещё и дать пользователю возможность самому это всё подгружать.

# DnD character creator
## Why
For the sake of a vintage, atmospheric geeky hobby experience. Also, for practicing the C language. And, of course, for the great and pure love of character sheets.\
In general, it might also have some minor practical use if you want to create characters during long trips or in remote places without internet access :)
## Core Functionality
This is still in the planning stages, of course, but still.\
**At the very least**, it's worth implementing the core character creation functionality with the selection of race, class, etc., naturally by adding automatic stat generation and equipment selection.\
**As long-term future plans**, add the capability to fetch information about classes and such from somewhere, and even give the user the ability to load all this data themselves.
